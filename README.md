# Airbnb Berlin — EDA & Market Analysis

Exploratory data analysis of 14,000+ Airbnb listings in Berlin using Inside Airbnb's public dataset.

## Tech stack
Python · Pandas · Matplotlib · Seaborn · Power BI (in progress)

## Project structure
| File | Description |
|---|---|
| `Airbnb_Berlin_Eda.ipynb` | Data loading, cleaning and preparation |
| `Airbnb_Berlin_Eda02.ipynb` | Exploratory analysis and visualisations |

## What it covers
- Full data cleaning pipeline: variable selection, type conversion, null handling
- Price distribution analysis and outlier treatment (95th percentile cap)
- Correlation heatmap across numerical variables
- Neighbourhood demand ranking and room type breakdown
- Power BI dashboard with interactive KPIs and market overview (in progress)

## Data source
[Inside Airbnb](http://insideairbnb.com/) — Berlin listings (public dataset)
